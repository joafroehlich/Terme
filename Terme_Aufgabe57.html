Berechnen Sie:

4x2 - 9a2 - 6ab - b2  

4x2 - (9a2 + 6ab + b2) 

9a2 + 6ab + b2 = (3a + b)2 

Zerlegung in Faktoren: 

4x2 - (9a2 + 6ab + b2) = 4x2 - (3a + b)2   

4x2 - (3a + b)2 ist aufgebaut wie das 3. Binom, 

a2 - b2 = (a + b)(a - b) somit 

4x2 - (3a + b)2 

= (2x - (3a + b))(2x + (3a + b))
