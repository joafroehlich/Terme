Berechnen Sie:

4x² - 9a² - 6ab - b²  

4x² - (9a² + 6ab + b²) 

9a² + 6ab + b² = (3a + b)² 

Zerlegung in Faktoren: 4x² - (9a² + 6ab + b²) = 4x² - (3a + b)²   

4x² - (3a + b)² ist aufgebaut wie das 3. Binom, a² - b² = (a + b)(a - b) somit 

4x² - (3a + b)² 

= (2x - (3a + b))(2x + (3a + b))
